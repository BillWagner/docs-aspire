ms.date: 07/22/2025

To install the Aspire CLI:

01. Open a terminal.
01. Run the following command to install the Aspire CLI:

    ### [Unix](#tab/unix)

    ```sh
    curl -sSL https://aka.ms/aspire/get/install.sh | bash
    ```

    ### [Windows](#tab/windows)

    ```powershell
    Invoke-Expression "& { $(Invoke-RestMethod https://aka.ms/aspire/get/install.ps1) }"
    ```
